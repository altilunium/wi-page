# wi-page
Rank Wikipedia Article's Contributors by Byte Counts.

## Live Demo
https://replit.com/@rtnf141/wi-page

## How To Use
`python3 wpage.py WikipediaArticleTitle` (id.wikipedia)\
`python3 wpage.py WikipediaArticleTitle en`(en.wikipedia)
